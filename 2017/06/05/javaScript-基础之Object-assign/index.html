javaScript 基础之Object.assign()

一：用途

方法用于将所有可枚举的属性的值从一个或多个源对象复制到目标对象。它将返回目标对象。
这里需要注意的有以下两点：

  • 自身可枚举属性
  • 返回目标对象

二：使用方法

  • 输入参数

即第一个参数为目标对象，后续的为源对象，源对象可以有一个或多个。
assign分配属性而不是复制或定义新的属性。如果合并源包含了 getter，那么该方法就不适合将新属性合并到原型里。
如何理解该描述，简单的来说就是源对象属性的描述符不会拷贝到目标对象。举个简单的例子：

对于对象obj1，其name属性设置了相应的描述信息，即该描述信息不会通过assign方法分配给obj2。
obj2的最终形式是{name:”test”}//即assign会调用源对象的getter方法，将值赋值给目标对象。

  • 返回数据
    返回值即为目标对象。
